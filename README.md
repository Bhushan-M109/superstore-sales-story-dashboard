# Superstore Sales Story Dashboard

## Overview
This project presents an interactive Tableau story dashboard analyzing Superstore sales data. It transforms raw retail data into meaningful business insights through structured visual storytelling.

## Objectives
- Analyze sales performance across regions and categories
- Identify top-performing products
- Understand customer trends
- Evaluate profitability patterns

## Dataset
The dataset contains retail transaction records including:
- Sales
- Profit
- Quantity
- Region
- Category
- Customer segments

## Features
- Interactive story navigation
- Insight-driven visualizations
- Business-focused analysis
- Clean dashboard design

## Tools Used
- Tableau
- Data Visualization
- Data Analysis

## Files Included
- Table's_Story.twbx → Packaged Tableau dashboard
- Superstore Dataset.xlsx → Source dataset

## Key Insights
- Certain regions generate higher revenue than others
- Profitability varies significantly by product category
- A small set of products contributes most revenue

## Use Case
This dashboard helps business stakeholders:
- Monitor sales performance
- Identify profitable segments
- Make data-driven decisions
- Improve strategic planning
